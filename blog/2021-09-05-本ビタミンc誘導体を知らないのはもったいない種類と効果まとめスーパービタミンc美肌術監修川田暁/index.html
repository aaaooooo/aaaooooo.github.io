【本】化粧品は成分で選ぶ！ビタミンC誘導体の種類と効果「スーパービタミンC美肌術」監修：川田暁

By あおな 05 September 2021

こちらの記事でも、ちょこっと書きましたが、上↑↑の本を読みました。

「美肌事典」（↑）を読んだ後、私の肌タイプ的にはビタミンC誘導体（スーパービタミンC）が相性の良い成分であることがわかったんですね。

【私の肌タイプと毛穴汚れタイプ】

  • 混合乾性肌（顔の中心部分は皮脂多め、頬など周辺は乾燥しているタイプ）
  • 黒ずみ毛穴（酸化した皮脂が毛穴に詰まって黒ずんでいるタイプ）

上記のような肌タイプにはビタミンC誘導体が相性が良いということが判明。

ビタミンC誘導体ってなんぞや、というところなんですが、基本的には美白化粧品に含まれている美白有効成分の一種です。

ひとえにビタミンC誘導体といっても、いろんな種類があるようで、自分の肌にあったものを見つけたいと思って、読んでみました。

ビタミンC誘導体の種類

まずは普通のビタミンCについて。

ピュアビタミンCに分類されるもの

  1. アスコルビン酸（ビタミンCの別名）
  2. アスコルビン酸Na
  3. アスコルビン酸K
  4. アスコルビン酸Ca など。 この他にも、色々種類があります。

こちらはビタミンC誘導体ではありません。
名前のまま、通常のビタミンCのことです。
即効性が高い分、水溶性で壊れやすく、浸透しにくいのが弱点。

また、ピュアビタミンCが何らかのきっかけで、**ビタミンCラジカル（悪いもの）**という成分に変わってしまう事があるみたいなんですね。
そうなると、肌に良いどころが、悪さをするので要注意。

コスメでは、ビタミンCラジカルになりにくいビタミンC誘導体（スーパービタミンC）が配合されているものを選んだほうが良さそう。

ビタミンC誘導体（＝スーパービタミンC）に分類されるもの

（　）内は化粧品で使われるの表記

  1. 水溶性タイプ
    　- SAP（リン酸アスコルビル３Na)
    　- MAP（アスコルビン酸リン酸マグネシウム）
    　- AA2G（アスコルビルグルコシド）
  2. 油溶性タイプ
    　- VC-IP（テトラヘキシルデカン酸アスコルビル）
  3. 両溶性タイプ
    　- APPS（パルミチン酸アスコルビルリン酸３Na）
  4. 万能タイプ
    　- GOVC（カプリリル２-グリセリルアスコルビン酸）

こちらの本で紹介されていたビタミンC誘導体一覧です。
（これ以外にもあると思います。）

このなかで気になったのは万能タイプのGOVCと、両溶性タイプのAPPS。

この２つの成分は下記の効果が期待できるそう。

GOVCとAPPSの効果

  • シミ、そばかすの予防・改善
  • しわ・たるみの予防効果
  • ニキビ予防・改善
  • 保湿効果
  • バリア機能維持

この本いわく、GOVCのほうが効果が高いようです。
しかし、このGOVCが入った化粧品が見つからないんですが、どういうことなんだろう？

結論として、

APPS配合の化粧品が良い！

ということで、私は次点で効果の高いAPPSが配合されているものと、混合肌向けのSAP、MAP、AA2Gが配合されているものをメインに、化粧品選びをしていくことに決定！

検索の基準は、以下の通り。

  • ビタミンC誘導体（特にAPPS）が多く配合されている。
  • ミネラルオイルが入っていない
  • お値段が２０００円前後のもの

ビタミンC誘導体が多く配合されているかどうかの判断は、成分表の記載位置で判断しました。
先頭近くに記載されていれば多め、後ろの方に記載されていれば少なめ、と判断。

ミネラルオイルは、毛穴づまりを起こす可能性があるということで、避けました。 なんてったって、毛穴づまりを解消したいわけなので。

お値段が２０００円前後のものを選んだのは、じゃぶじゃぶ使いたいから。お高いと、ちびちび使ってしまって、効果が実感できなさそうなので（笑）

ということで、調べに調べ尽くして見つけたのがこちら。

まずはロロシューン VCクリアコントロールローション。

この化粧水は、ビタミンC誘導体（APPS）以外にも、複数のビタミンC誘導体を含んでいるのと、ミネラルオイル（鉱物油）フリー、お値段も１５００円（Amazon価格）なので、じゃぶじゃぶ使えそう。

いろんな成分が入っているんですが、界面活性剤も一番安全性の高いものが使われていたり、ビタミンC誘導体の弱点である乾燥についても、保湿剤でカバーしてくれているというところが、すばらしい！

同じメーカーで乳液なども出ているんですが、成分的にはこちらの化粧水タイプが一番良かったです。

化粧水はまだ家の在庫がかなりあるので、それを使い切ったら、こちらを使ってみたいと思っています。

と、思っていたら、まさかの楽天スーパーセールで、1500円が1000円になっている！

＼500円OFF／化粧水 ビタミンC誘導体 ビタミンC【VCクリアコントロールローション 120ml】 毛穴 ニキビ APPS 美容液 黒ずみ 送料無料 ローション アプレシエ 化粧水 保湿 脂性肌 敏感肌 乾燥肌 開き 無添加 保湿 くすみ 美肌 透明感 潤い ／ロロシューン
価格：1000円（税込、送料無料) (2021/9/5時点)

楽天で購入

思わず、ポチってしまいましたよ（笑）

使用感など、レビューはまた後日！

お次はこちら。

Bionist（ビオニスト) 高濃度 ビタミン 美容液 美肌菌 乳酸菌 生産物質 高浸透 ビタミンC（APPS） 10mL ３週間用 日本製

こちらもAPPSの配合で、かつミネラルオイルフリー。
量が少ない割にお値段が２０００円ほどするので、そこが難点ですね。
一見少ないんですが、この量で３ヶ月分らしいので、選出しました。

その他の成分も、かなり安全性が高いものばかりだったので、安心して使えそう！
ご褒美的に使いたい美容液です。

お次はこちら。

3種類のビタミンC誘導体 (浸透力100倍) 美容液 エイジングケア 日本製 40ml

こちらは化粧水も出ていて、そちらのほうがいいな〜と思ったら、Amazonでは売り切れ。
美容液はまだ売っているようでした。
４０gで２０００円ほどなので、そこがやや難点ですね。

こちらも複数のビタミンC誘導体がはいっているのと、ミネラルオイルフリー。
あとは、保湿剤がしっかり入っているのと、界面活性剤も安全性の高いものが使われていて、良さそうと判断しました。

お次はこちら。

Nanarobe (ナナローブ) スーパー VC ローション 化粧水 150mL 約2～3ヵ月分 ［ ツバメの巣 コロカリア ビタミンC スキンケア ］

こちらはAPPSのを配合しつつ、エイジングケアにも対応しているタイプの化粧水。
なんとなくフェミニンなパッケージが好きになれないところが難点ですが、（←どうでもよい笑）、抗酸化作用のあるトコフェリルリン酸Ｎａや、保湿力のあるツバメ巣なども配合していて、色々カバーしてくれてます。

保湿もしてくれて、酸化からも守ってくれて、かつビタミンC誘導体も入っているということで、盛りだくさんで嬉しい化粧水。

１５０ml２５００円ほどなので、お値段を気にせずじゃぶじゃぶ使えそうということで選出しました。

以上、私が調べたAPPS配合の化粧品たちです！
あくまで私（混合乾性肌、黒ずみ毛穴タイプ）に合いそうな化粧品ということで、ご理解ください。
乾燥肌の方には、ちょっと合わない可能性があります。
口コミなどでも、肌が乾燥する、といったコメントがちらほらあるので、もし使うときは、しっかり保湿するのをおすすめします。

余談。

なんだか自分にある化粧品を探すだけなのに、ものすんごい時間がかかりました、、、、（汗）　　

化粧品を成分で選ぶのって、結構大変なんだな〜と自分でやってみてしみじみ実感。
APPSが入っていて良さそうと思ったら、ミネラルオイルが入っていたり、あまり良くない界面活性剤がはいっていたりで、思った以上に難航。

美白化粧品でもビタミンC誘導体が入っていないものもあるので、検索の仕方が難しいですね。

iherbでも検索してみたんですが、私の検索の仕方が悪かったのか、全然該当するものがありませんでした。
（iherbで良さそうなものがあったら、是非教えてほしいです！）

ということで、長くなってしまいましたが、ここまで読んでいただき、ありがとうございました！！
